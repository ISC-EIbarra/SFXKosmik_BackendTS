## SFX-KOSMIK (Node + Express (TS))

> Proyecto web con node enfocado en la venta de cursos de maquillaje.
> Implementa Node + Express con TypeScript para el backend.

<br>

**Acciones**

```
- Creación de cuentas a usuarios.
- Recuperación de cuenta.
- Comprar un curso.
```

**Tecnologías**

```
- Node JS
- Express
- MongoDB
- TypeScript
```
